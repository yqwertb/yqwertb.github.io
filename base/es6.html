# ES6

# 1 var let const

var和let都定义一个变量，而const定义一个常量

var会造成变量提升，let和const声明的变量/常量只存在于当前块级作用域

let和const都会造成暂时性死区

# 2 暂时性死区是什么

function() {
    console.log(tmp); //ReferenceError
    let tmp = 1;
}

在一个块级作用域中，let，const声明变量/常量之前的部分代码不能使用let，const声明的变量/常量。

# 3 箭头函数和普通函数的区别

  • 如果只有函数内部只有一个表达式，既可以省略return和花括号
  • 箭头函数不能作为构造函数
  • 箭头函数没有自己的this，它会获得自己在定义时所处的块级作用域中的this，并继承。如果没有就继续往上找。
  • 箭头函数继承的this不能改变，call、apply、bind 无法改变箭头函数的this。
  • 箭头函数没有自己的arguments对象，箭头函数中访问arguments是获取上层块级作用域的arguments
  • 箭头函数没有prototype属性
